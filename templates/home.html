AI-Powered Telephony Agent for Flat and Property Queries

This Django project answers property questions using database facts and supports API, Twilio voice webhook, and Vapi webhook flows. Open the links below to present the project cleanly instead of landing on a 404 page.

Running with provider: {{ provider }} and default property ID: {{ default_property_id }}

Browser Endpoints

  • /api/health/ checks service status
  • /api/properties/ lists property data
  • /api/transcripts/ lists saved conversations
  • /admin/ opens Django admin

Important API Routes

  • POST /api/ask/ for property Q&A
  • POST /api/twilio/voice/ for Twilio voice webhooks
  • POST /api/vapi/webhook/ for Vapi callbacks

Quick Demo Questions

  • What is the price?
  • What is the carpet area?
  • Is parking available?
  • What is the location?

Tip: keep python manage.py runserver running in one terminal, then test the JSON and webhook endpoints from a second terminal.
